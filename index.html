Ranking Entities for Web Queries

DWS Group, University of Mannheim

Ground truth dataset

REWQ Robust04 Dataset

  • REWQ-Robust04: entity relevance (trec qrel format)
  • REWQ-Robust04: document entity mentions

REWQ Clueweb12 Dataset

  • REWQ-Clueweb12: entity relevance(trec qrel format)
  • REWQ-Clueweb12: document list (retrieved with SDM)
  • REWQ-Clueweb12: FACC1 document entity mentions are available from the Lemur Project
We provide only our own annotations for the DBpedia/Wikipedia entity relevance here.

Ranking run data

  • Run files for Robust04
  • Run files for Clueweb12
